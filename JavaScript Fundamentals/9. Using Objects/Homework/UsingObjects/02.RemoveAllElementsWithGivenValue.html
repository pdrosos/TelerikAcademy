Remove all elements with given value

Write a function that removes all elements with a given value. Example:
propertyvar arr = [1,2,1,4,1,3,4,1,111,3,2,1,'1'];
arr.remove(1); //arr = [2,4,3,4,111,3,2,'1'];
Attach it to the array type. Read about prototype and how to attach methods.
