Main Menu

When the game is started, you are taken to the main menu:

From there you may choose which game you wish to play. Just type the corresponding number and press Enter to start the game.

You have 1 minute for each game. After the time is up, you see your score. Press a key to be taken back to the Main Menu.

Each game has its own "rules" screen, where you can find out the controls and rules for the corresponding game.

Your high scores from each game are displayed in the Side Bar.

Black Hole Game

Pavlina Drosos /a.k.a. pavlinadrosos/


Do you pride yourself on your memory. Well it's time you prove it. Spot which form is missing after the Black Hole. Select the corresponding number from the list.

Test Your Reflexes Game

Antoaneta Marinova /a.k.a. antoaneta.marinova.5/


Can you spot the letter among the distractions and type it fast enough to get a bonus. Try your reflexes with this game, but beware as the distractions will increase with each try.

Mathematics Game

Ivan Dimitrov /a.k.a Ivan.Dimitrov.bg/


Are you good with Math? Well we will test your calculus abilities in the Mathematics game. Solve the equations as fast as you can and type your answer. Remember that you only have a minute to impress us with your skills.

Test Your English Game

Katalina Dimitrova /a.k.a lina94/


If you understand this documentation, you may not have a problem with this game. But every cat needs to know its English and you only have 1 minute to type in as many correct answers as you can!

Action Game

Emilia Toteva /a.k.a emilijohnson/


Tired of being tested? Well this is the place to unwind. Select your difficulty level and your weapon and make sure you don't get eaten by a fish. Remember, each weapon has its own range and reload time, so choose wisely.

Use the arrow keys to move and space to fire.

Classes

The following chapter describes the classed we used in out game project.

Action Game

The class contains the main game logic of the Action Game.

Felix

The class contains the played character object in the Action Game. This includes position, checking if the position is empty and moving the character.

Fish

The class contains the fish object in the Action Game.

Weapon

The class contains the weapon object in the Action Game.

BlackHole

The class contains the black hole object with its shape and coordinates.

BlackHoleGenerator

The class generates the black hole object.

Figure

The class contains falling object with its shape and coordinates in the Black Hole game.

FiguresGenerator

The class generates random figure objects and lists with random figure objects in the Black Hole game.

Game

The class contains the main game logic of the Black Hole game.

StartGame

The class contains the Black Hole game's entry point.

Window

The class prints all game elements of the Black Hole Game.

Mathematics

The class contains the Mathematics game.

EnglishTest

The class contains the English Test game.

ReflexesGame

The class contains the Test Your Reflexes game.

Menu

The class contains the Main Menu of the game.

Methods

The following chapter describes the methods we used in out game project.

Not all methods are listed in the documentation.

SideBar()

This method is used to draw the Side Bar of the main menu, as well as, the side bars in each games.

GameStory()

This method is used to print the game story in the main menu.

SelectedGame()

This method is used to start the different games from the main menu.

FelixPicture()

This method is used draw Felix's picture in the main menu.

Play()

This method contains the game logic of several of the games and is used to call games from the SelectedGame() method.

TimeIsUp()

This method handles what happens when the Timer has reached its elapsed time.

TrueOrFalse()

This method is used in the Test Your English game to handle correct and incorrect answers.

PrintAtPosition()

This method is used to print different forms and structures in the games.

YouAreWrong()

This method handles the console window behavior if the answer is wrong in the Test Your Reflexes Game.

GetBlackHoleSymbols()

This method generates the black hole's shape in the Black Hole game.

GenerateBlackHole()

This method generates a new Black Hole object.

GetRandomList()

This method is used to generate a random list with objects with random shape and color for the falling figures in the Black Hole game.

PrintFigure()

This method is used to print the falling figures in the Black Hole game.

IsGameOver()

This method is used in the Black Hole game to check is the time for the game is over.

SelectDifficulty()

This method is used to select the difficulty level of the Action game.

SelectWeapon()

This method is used to select the weapon in the Action game.

WeaponKind()

Sets a different character for ActionGame.CharInfo weapon, depending on the selected weapon.

PrintMatrix()

This method is used to print the matrix in the Action game.

ClearMatrix()

This method is used to clear the matrix in the Action game.

NumberOfFishes()

This method regulates the number of fishes to be printed in the Action Game.

GameEnd()

This method is called at the end of the Action Game; it displayed the current and the high scores.
